DirectX11 Shaders

I have experience with graphics programming including both DirectX11 (HLSL) and OpenGL. I've created a shader for each stage of the graphics pipeline from the vertex shader all the way through to the pixel shader.

During development in DirectX11 I created an application that showcased shadows, post processing, multiple lights, vertex manipulation, dynamic tesssellation and more. I am most proud of my implementation of dynamic tessellation as it is a fundamental graphics programming technique used in games to balance performance and visual fidelity. My implementation was based on the position of the camera so as the player gets closer to a piece of geometry the more detailed the geometry will become.

I used ImGui throughout in order to assist development. ImGui is a graphical UI which can be used to modify parameters during runtime. This improved development time as I can test various values without having to retype a line of code and recompile each time a new parameter needs tested.
