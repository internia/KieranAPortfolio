Inventory System (UE4)

I developed an inventory system that allows the user to pick up usable items in the game world and places them into the players inventory for later use/functionality. The mechanic was loosely inspired by popular RPG games like World of Warcraft and Skyrim, however the mechanic can be applied to a wide range of genres including but not limited to point and click adventure games like Monkey Island, side scrolling platformers/sandboxes like Terraria and many more.

Inventory systems are an element of game development that transcends genre as the aesthetic and functionality can be changed but the systems logic largely stays the same. The system is made up of several components and was achieved by using a mixture of C++ scripting for functionality and blueprinting for UI elements within Unreal Engine 4

A demonstration of the inventory system can be seen here: https://www.youtube.com/watch?v=pw-5VKCHDiM
