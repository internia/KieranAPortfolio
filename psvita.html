Playstation Vita Game

The purpose of the application was to showcase how the box2d physics engine can be used to create a 2.5D physics based game. The application was programmed in C++ and was created using the Playstation Vita development kit. The application is a physics based game which takes influence from games such as Angry Birds. The player aims and fires a projectile into a structure made up of many dynamic bodies. When the projectile makes contact with the dynamic bodies the force causes the structure to fall depending on where the force is made. The player must aim to knock every enemy inside the structure onto the floor. If the player successfully knocks all three enemies the player wins, is greeted with a splash screen and then the game is over.

The application utilizes a number of different classes to achieve the final build. One of the fundamental classes is the shape manager class. This class is a level editor, it allows the easy creation of dynamic bodies, using just one line of code and four parameters it can create a shape anywhere in the level. The class is set up in this way as it makes the level easy to maintain and add extra functionality where needed. Another important class was the HUD class created to indicate to the player how charged up their shot was, creating some feedback for the player.
